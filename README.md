📱 BMI Calculator App (Flutter)
--
A simple and user-friendly BMI (Body Mass Index) Calculator built using Flutter.
This app allows users to input their weight and height to calculate their BMI and determine their health category.

🚀 Features
---
* 🧮 Calculate BMI based on weight and height
* 📏 Supports height in feet and inches
* 📊 Displays BMI result with category:

  * Underweight
  * Normal Weight
  * Overweight
  * 📱 Simple and clean UI
  * ⚡ Fast and responsive

---

 🛠️ Tech Stack
--
* Flutter
* Dart
* Material UI Design

---

📊 How It Works
---
1. Enter your - Weight (kg)
2. Enter your - Height (feet & inches)
3. Click on - Calculate BMI
4. Get your BMI result instantly


📌 BMI Formula
---
[
BMI = \frac{weight (kg)}{height (m)^2}
]
